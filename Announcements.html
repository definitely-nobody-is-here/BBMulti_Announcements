Multi-game Servers will Not be Added

(Issue #70)
After reviewing this suggestion, we have decided that giving people the ability to run multiple games on one server would be too detrimental to the server's performance. Running multiple servers would be more resource-efficient and better for performance. It's why you have the ability to run multiple servers: To run multiple games. So, we will not be adding that feature. We hope you continue suggesting things to add and we will continue to look at them.

5/3/21 - Critical Bug Discovered (Issue #25)

A critical bug affecting gameplay was found today, making it impossible to play with more than five players. The bug has an unknown origin and is currently being worked on. The issue affects map loading: The map cannot load and will be blank on all client screens until less than six people are connected. Possible causes include: lack of port bandwidth restricting the server from sending the map, clients not being able to render the map when there are too many players, conflicts in the code causing inability to load maps, inefficient code. For now, refrain from playing with 6 or more players.
